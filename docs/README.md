# Drafter — Documentation

Drafter is a layered architecture diagramming tool with built-in AI and security analysis features. Use it to design systems, drill into sub-components, model threats, and assess your security posture.

## Guides

| Guide | What it covers |
|-------|---------------|
| [Getting Started](./getting-started.md) | Sign in, create a project, and draw your first diagram |
| [Diagrams & Canvas](./diagrams-and-canvas.md) | Nodes, edges, grouping, layers, copy/paste, undo/redo |
| [AI Assistant](./ai-assistant.md) | Generate diagrams, evaluate architecture, contextual Q&A |
| [Trust Boundaries](./trust-boundaries.md) | Mark trust zones on your DFD for accurate threat modeling |
| [STRIDE Threat Analysis](./stride-threat-analysis.md) | Run AI-powered threat analysis and manage findings |
| [Security Posture Score](./security-posture-score.md) | Compute a 0–100 security health score for your architecture |
| [Attack Mind Simulator](./attack-mind-simulator.md) | Simulate red-team attack paths through your architecture |
| [Threat Dashboard](./threat-dashboard.md) | Manage, filter, and export all threats across a project |
